## 1.1 集合的理解和好处

数组的不足之处：

- 长度开始时必须指定，而且一旦指定，不能更改；
- 保存的必须为同一类型的元素；
- 使用数组进行增加/删除元素的示意代码——比较麻烦。

集合的好处：

- 可以动态保存任意多个对象，使用比较方便；
- 提供一系列方便的操作对象：add、remove、set、get等（增删改查）；
- 使用集合添加，删除新元素的示意代码——简洁。
